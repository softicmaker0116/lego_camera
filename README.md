# lego_camera
*#Arduino* *#LinkIt* *#Camera* *#Lego*
> This project is mainly aimed at designing and implementing a lego camera 
> with open source microcontrollers and hardware components.
# 
##### Components:
* Arduino Mini or LinkIt Smart 7688 Duo
* CJ-OV528 camera
* SD card
* LED
* button
* Lego
* battery
